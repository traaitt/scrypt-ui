Main information
URL: POOL_STRATUM_ADDRESS_AND_PORT
One of the stratum addresses above in the Pool Configuration section depending on the difficulty.
Example: stratum+tcp://sha256.ae:1000

Wallet_address: YOUR_LTC_WALLET_ADDRESS-YOUR_DOGE_WALLET_ADDRESS.WORKER_NAME
Your valid wallet address for payouts. Generate it using the official wallet or use the address of any crypto exchange.
Example: HIVGVHHGUTFYCTGVHBJ

RIGId - Optional name of the worker for statistics. Use English letters, numbers, and symbols '_' or '-'.
Example: SCRYPT-ASIC-SHA256-440_500

Password can be left blank.
